Objectives

Upon completion you should be able to:

  • Understand what JavaScript is
  • Identify where to insert JavaScript code in HTML pages
  • Understand the purpose of variables
  • Understand what type of variables JavaScript supports
  • Be comfortable with all JavaScript operators
  • Understand the purpose and uses of arrays
  • Manipulate arrays and array elements
  • Understand the purpose and uses of objects
  • Manipulate object properties
  • Write your first JavaScript code

Homework

Write a JavaScript page that outputs the following:

  • A multiplication table for 10 x 10
  • Write this output within an HTML table
  • Add column headers and row headers using a different color
  • Do not use any Javascript commands such as variables or looping techniques. Simply write multiple document.write( )
  • Output should look like the following:
X 1 2 3 4 ...
1 1 2 3 4 ...
2 2 4 6 8 ...
3 3 6 9 12 ...
4 ...
  • Create 2 variables
  • Initialize each to a any numeric value
  • Muliply these two variables and output the variables and result
  • Create and load an array
  • Output the 3rd element
  • Create and load an object
  • Output one of the properties
